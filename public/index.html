Набор команд для создания react-приложения:

  1. pwd - проверяем текущую дирикторию
  2. npx create-react-app . - создаём react-приложение: инициализируется package.json. B index.html вставляем скрипт index.js
  3. npm start - запускаем веб-сервер
  4. npm install --save-dev prop-types - устанавливем библиотеку для проверки/отслеживания типов пропсов (props)
  5. npm install --save-dev prettier eslint husky - устанавливем библиотеки для форматирования и проверки качества кода
  6. npx mrm@2 lint-staged - эта команда Она установит и настроит husky и lint-staged в зависимости от инструментов качества кода из зависимостей проекта в package.json
